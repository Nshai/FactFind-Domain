# Product Overview

## IntelligentOffice Suite

IntelligentOffice is a comprehensive financial services platform consisting of 12 packages that provide CRM, fact-finding, employment management, and financial planning capabilities for financial advisors and their clients.

### Core Components

**IntelligentOffice (Legacy .NET Framework)**
- Main web application and business services
- Client portal and financial planning dashboard
- Employment, fact-finding, and CRM functionality
- Built on .NET Framework 4.7.2 with NHibernate ORM
- Multi-tenant architecture supporting various financial institutions

**Monolith.FactFind (.NET 8)**
- Modern fact-finding service built on .NET 8
- RESTful API for employment and client data management
- Microservice architecture with comprehensive testing
- Swagger/OpenAPI documentation

### Key Features

- **Fact Finding**: Comprehensive client data collection and management
- **Client Portal**: Web interface for clients to view and manage their financial information
- **Multi-tenant Support**: Serves multiple financial institutions with isolated data
- **Integration Ready**: APIs for external systems like Vision/Multiply integrations

### Target Users

- Financial advisors and planners
- Financial services institutions
- Clients accessing their financial information through portals
